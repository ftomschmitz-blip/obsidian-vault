Hermes - Obsidian - Claude Setup Architektur

Interaktives & autonomes Wissensnetzwerk (Lokal & VPS-Server gekoppelt)

Säule 1: Git-Sync

  • • Mac-Seite: Obsidian Git Plugin committet & pusht im Hintergrund alle 15 Minuten zum privaten GitHub Repo.
  • • VPS-Seite: Ein Hermes-Cronjob holt (pull) Änderungen alle 30 Minuten mit git pull --rebase.
  • • Effekt: Volle Synchronität deiner Gedanken auf allen Geräten und dem Server ohne Cloud-Kosten.

Säule 2: Lokaler Claude MCP

  • • Local REST API: Gibt deinem lokalen Obsidian eine sichere HTTPS-Schnittstelle.
  • • Claude Desktop: Kann direkt über das mcp-obsidian Plugin lokale Dateien lesen, durchsuchen und anlegen.
  • • Nutzen: Direkter Zugriff auf dein Second Brain ohne Hochladen von Texten in Serverumgebungen.

Säule 3: Remote SSH-Bridge

  • • Die Bridge: Kettelt deinen lokalen Claude Desktop direkt mit dem remote Hermes Agent.
  • • Befehl: Startet im Hintergrund auf dem VPS hermes mcp serve über einen sicheren SSH-Tunnel.
  • • Nutzen: Claude kann remote Befehle ausführen, Recherchen auf dem VPS anstellen und deine Server-Umgebung steuern.
